    [BurpSuite] 다크테마 적용하는 방법

    서론

    Burpsuite 최초 설치 후 실행하면 하얀 배경화면이라
    계속 보고 있으면 안구 건강에도 좋지 않다.

    Dark Theme 적용

    Burpsuite 기본 설정으로 테마를 바꿀 수 있다.

    Dark Theme 설정

    1. User options 탭
    2. Display 탭
    3. Theme를 Light > Dark로

    더 많은 테마 사용하기

    Burp Customizer를 이용해서 다양한 테마를 적용할 수 있다.
    하지만 튜닝의 끝은 순정.
    기본 옵션의 Dark Theme 사용 중.

    스크린샷

    하지만 문서에 스크린샷 삽입 시 검정 배경은 좋지 않다.
    그럴 때는 많은 사람들이 사용하는 픽픽 등의 도구를 이용하여 색을 반전 시키면 된다.

    색반전

    1. Ctrl + A 혹은 반전 시킬 부분 선택
    2. 홈 > 효과
    3. 색반전

    결론

    테마 변경

    • User options > Display > Theme 변경
    • 다른 테마 사용 시 Burp Customizer 사용

    문서 작성 시

    스크린샷 삽입 시에는 색을 반전시키자.
