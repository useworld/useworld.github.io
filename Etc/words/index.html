    [기타] 보안/네트워크 기본 용어

    서론

    이 동네는 유독 약어를 많이 쓴다.
    보통 모르고 쓰는 경우가 많아 자주 쓰는 용어를 모았다.

    용어 정리

    분야와 상관 없이 알파벳 순으로 정렬했다.

    • ACL : Access Control List
    • APT : Advanced Persistent Threats
    • ARP : Address Resolution Protocol
    • ASLR : Address Space Layout Randomization - PE파일이 메모리에 로딩될 때 로딩 시작 주소 값을 랜덤하게 바꾸는 기술.
    • AWS : Amazon Web Services(Cloud)
    • C&C, C2 : Command & Control
    • CC Attack : Cache-Control Attack
    • CIDR : Classless Inter-Domain Routing - '사이더'라고 읽는다.
    • CLI : Command-line Interface
    • CVE : Common Vulnerabilities and Exposures
    • CWE : Common Weakness Enumeration
    • DAC : Discretionary Access Control
    • DDoS : Distributed Denial of Service
    • DGA : Domain Generation Algorithm
    • DHCP : Dynamic Host Configuration Protocol
    • DMZ : DeMilitarized Zone
    • DNS : Domain Name System
    • DoA : Degree of Acceptance - 위험 정도를 측정한 뒤 ‘수용할 수 있는 수준’ 이상의 위험에 대해서만 비용을 들여 대응하는 것으로 결정하는데, 이 ‘수용할 수 있는 수준’을 보안 분야에서는 DoA(Degree of Acceptance)라고 한다.
    • DRM : Digital Rights Management
    • EDR : Endpoint Detection and Response
    • ESM : Enterprise Security Management - 이종 장비간의 패킷 로그를 통합하여 관리, 알람
    • GRE : Generic Routing Encapsulation
    • GUI : Graphical User Interface
    • HIDS : Host-Based IDS
    • ICMP : Internet Control Message Protocol
    • ICS : Industrial Control System - 산업 제어 시스템
    • IDC : Internet Data Center
    • IDS : Intrusion Detection System
    • IIS : Internet Information Services
    • IPS : Intrusion Prevention System
    • ISMS : Information Security Management System
    • ISP : Internet Service Provider
    • LLCF : Link Loss Carry Forward
    • MAC : Mandetory Access Control
    • MFT : Master File Table
    • NAT : Network Address Translation
    • NIC : Network Interface Controller
    • NIDS : Network-Based IDS
    • NMS : Network Management System
    • NTP : Network Time Protocol
    • PAT : Port Address Translation
    • PE : Portable Executable
    • PIMS : Personal Information Management System
    • PM : Pre-Maintenance, Prevention Maintenance
    • PMS : Patch Management System
    • POC : Power Of Community International Security Conference, Proof Of Concept
    • PUP: Potentially Unwanted Program
    • RBAC : Role Based Access Control
    • RDP : Remote Desktop Protocol
    • RegEx : Regular Expression - 정규표현식
    • SIEM : Security Information & Event Management - 이종 장비간의 패킷 로그를 통합, 분석하여 위험도 책정, 알람
    • SNMP : Simple Network Management Protocol
    • SOC : Security Operation Center
    • SSDP : Simple Service Discovery Protocol
    • SSL : Secure Socket Layer
    • TCP : Transmission Control Protocol
    • UDP : User Datagram Protocol
    • UTM : Unified Threat Management
    • VDI : Virtual Desktop Infrastructure
    • VPN : Virtual Private Network

    결론

    뭔지는 알지만 정확한 뜻을 모르는 경우가 많은데
    사람들은 안궁금한가 보다.
    잘 모를 때마다 구글링이 답이다.
    유식한 척하려면 약자의 의미정도는 알아줘야 한다.
    S사 개발 시 자주 보이는 SR 약자 뜻 정확히 아는 사람 얼마나 될까.
