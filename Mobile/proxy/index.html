    Proxy 설정 시 속도를 빠르게

    서론

    모바일에서 Proxy 설정 후 PC에서 Burpsuite 같은 도구를 활용하여
    진단 시 원래의 앱 동작 속도에 비해 엄청나게 느려진다.
    아마 요청과 응답이 느려져 그런 것 같은데 속터진다.

    일반적으로

    보통 같은 와이파이 AP에 모바일과 PC를 연결하여
    모바일에서 Proxy 설정 후 작업을 진행하게 된다.
    이러면 Proxy 설정 전과 비교할 때 답답할 정도로 느려진다.

    빠르게 하려면

    PC에서 핫스팟을 켠 후 모바일에서 접속한다.

    1. PC에 인터넷이 가능한 랜선을 직접 연결
    2. Windows에서 핫스팟 설정
    3. 모바일을 핫스팟 Wi-Fi에 연결
    4. Proxy 설정

    핫스팟이란?

    인터넷에 접속된 스마트폰이나 태블릿을 휴대용 공유기로 만드는 것을 말한다.
    모바일 데이터 통신망 또는 다른 Wi-Fi, 랜선으로 연결된 인터넷을
    자신의 기기를 통해 다른 기기와 공유한다.
    결국 자신의 기기를 휴대용 공유기로 만드는 기능이다.

    핫스팟 설정

    OS에 따라 다르지만 Windows 11의 경우 핫스팟 설정 페이지에서 모바일 핫스팟을 켜고
    네트워크 속성에서 Wi-Fi 이름과 비밀번호를 설정하여
    모바일에서 해당 이름의 Wi-Fi에 비밀번호를 입력 후 접속하면 된다.

    왜 빨라질까

    동일 공유기의 Wi-Fi 연결 시 다음의 과정을 거칠 것 같다.

    요청 시

    모바일 > 공유기 > PC(Proxy) > 공유기 > 외부

    응답 시

    외부 > 공유기 > PC(Proxy) > 공유기 > 모바일

    핫스팟 설정 후 직접 모바일과 진단PC를 연결하게 되면
    공유기를 거치는 과정이 줄어들어서 그런 것이 아닐까 추측된다.
    (아닐 수도 있다.)

    결론

    핫스팟을 사용할 수 없는 환경이 대부분이긴 하겠지만
    만약 제약이 없을 경우 꼭 한번 해보길 바란다.
    속도가 신세계다.
